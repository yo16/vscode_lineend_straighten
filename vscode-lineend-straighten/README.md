# vscode-lineend-straighten README

This VisualStudioCode extension aligns the line endings of the selected lines.
----
## Usage
1. Select lines.
2. Ctrl + K

before
```
abc
defghijk
zzzzzzzzzzz
xx
yyyy
```

 (Select & Ctrl+K)

after
```
abc             (line end)
defghijk        (line end)
zzzzzzzzzzz     (line end)
xx              (line end)
yyyy            (line end)
```
Line endings are neatly padded with spaces.
